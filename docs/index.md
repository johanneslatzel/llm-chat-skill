# Overview

Skill loading tools for the [llm-chat](https://github.com/johanneslatzel/llm-chat) ecosystem. Skills are reusable instruction sets stored as `SKILL.md` files with YAML frontmatter.

- **[Getting Started](examples/quick-start.md)** — install and load your first skill
- **[Tools](tools/index.md)** — `load_skill`, `get_skill_resource`, `set_skill`, `delete_skill`, `set_skill_resource`, `delete_skill_resource`
- **[Packages](packages.md)** — `SkillToolPackage` for `ToolSuite` integration
- **[Registry](registry/index.md)** — `SkillRegistry`, configuration, skill file format, `listing()`
- **[Environment Variables](env.md)** — configuration reference
- **[Architecture](architecture.md)** — high-level design
